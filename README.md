# Fly AI Practice Codes

Practice codes for Fly AI

## Local development (Windows)

```sh
py -m pip install --user virtualenv
```

```sh
py -m venv env
```

For access issues

```sh
Set-ExecutionPolicy Unrestricted -Scope Process
```

```sh
.\env\Scripts\activate.ps1
```

## Installing libraries

```sh

```
